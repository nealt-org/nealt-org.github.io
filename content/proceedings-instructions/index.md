If you want to publish a volume in the NEALT Proceedings Series, please send a request for publication to the [NEALT editor-in-chief](/contact/) in advance.

## General Information

- Published proceedings must consist of **peer-reviewed** articles, and must document the composition of the program committee, the review procedure, and the acceptance rate in the preface.
- Proceedings will be published by **University of Tartu Library** and digitally archived by [Tartu D-Space](https://hdl.handle.net/10062/4117).  They have persistent identifiers in the form of **Handles**, and can optionally be assigned **DOIs** as well.
- The frontmatter of the proceedings should clearly refer to "NEALT Proceedings Series", the volume number within the series, as well as the ISSN 1736-8197 & eISSN 1736-6305.

## Process

We can assign a **volume number** and create an **ISBN** once we have the following basic metadata:
- full list of editors
- full proceedings title
- planned publication date

For publication, we need:
- PDFs for each article in the proceedings + the full proceedings volume
- a file with metadata for each article

The metadata can be prepared in **any format you like** (e.g., Excel sheet, CSV, ...) and must, at a minimum, specify **author(s)** and **title** for each article.  Additional metadata fields, such as abstract and keywords, can also be provided and will appear on the "full item page" of the article on D-Space ([see an example here](https://dspace.ut.ee/items/5edf1ecc-b0ef-4d76-991d-521c22f9f30b/full)).

Please also indicate the **license** under which the proceedings should be published; D-Space defaults to [CC BY-NC-ND 4.0](https://creativecommons.org/licenses/by-nc-nd/4.0/), but we recommend using [CC BY 4.0](https://creativecommons.org/licenses/by/4.0/) if possible, which is the same license that e.g. \*ACL proceedings use.

## Frequently Asked Questions

### How far in advance should we send the material?

Ask for the series number and ISBN a few weeks in advance.  D-Space have historically been very fast in handling our requests, so it normally suffices to send the full materials a few days before the planned publication date.

### Can we get DOIs in advance, to include them in the PDFs?

Unfortunately, no.  D-Space can only generate DOIs _after_ the articles have been uploaded.  If you insist on showing the DOIs in the PDFs, we can achieve this in the following way:

1. D-Space publishes the PDFs (_without_ DOIs) and then generates DOIs.
2. You update the PDFs accordingly to include the DOIs and re-share the updated materials.
3. D-Space re-uploads the updated PDFs.

### Can we have a printed version of the proceedings as well?

We currently do not have a publisher for print versions of proceedings.  If you want to produce a small print run yourself, note the following:

- University of Tartu Library **_cannot_** act as the publisher for a printed edition.
- ISBNs are specific to the publication format, so the ISBN for the digital edition **_cannot_** be re-used for a printed edition.
- NEALT _can_ issue a separate ISBN for a printed edition, with just "Northern European Association for Language Technology" as the publisher.

As NEALT is officially registered in Sweden, we might be required to send a [legal deposit to the National Library of Sweden](https://www.kb.se/pliktleverans-och-isbn/eng/information-and-guides/what-is-legal-deposit.html) for print editions.  Please make sure to discuss this with the editor-in-chief in advance to avoid running into unexpected problems.


### Do we need to use a specific template for our proceedings?

NEALT imposes no formatting or layout requirements.
